Tweets

Type in a keyword you want to find tweets about, then pick the number of tweets you want to load. If you don't type in a keyword, tweets from around the RPI area will appear, but you still have to choose a number of tweets.

Clicking "Load" will grab the tweets from Twitter, but they will not appear on the screen. If you pick a file type and click "Export", then the loaded tweets will be placed in the respective file.

Clicking "Build" will build a Mongo database with the tweets that were loaded in it. If you do not click "Load" before "Build", there will be no tweets in the database.

Clicking "Read" will display the tweets from the database onto the screen.

If you type in a file name and click "XML", the tweets from the database will be loaded into an XML file with the file name you chose.

Every time you want to build the database and read from it, you have to refresh this page.
